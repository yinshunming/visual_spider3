# Visual Spider E2E Manual 验收清单

本文档列出了不适合自动化测试、需要人工验收的场景。

---

## M-01: 可视化配置体验

### 验收场景
点击元素生成选择器的交互体验

### 验收要点
1. **CDP 选择器生成准确性**
   - 在可视化配置页面，点击页面元素应能正确生成 CSS/XPath 选择器
   - 选择器应能准确匹配目标元素

2. **选择器手动调整体验**
   - 手动调整选择器时应响应流畅
   - 调整后的选择器应能正确高亮目标元素

3. **元素高亮显示**
   - 选中元素时应有清晰的高亮显示
   - 高亮不应遮挡目标元素内容

### 验收步骤
1. 进入任务配置页面
2. 选择"列表页模式"
3. 输入测试 URL 并加载页面
4. 点击"选择容器"按钮
5. 点击页面中的列表容器
6. 验证生成的 containerSelector
7. 点击"选择链接"按钮
8. 点击列表中的文章链接
9. 验证生成的 itemUrlSelector
10. 切换到内容页配置
11. 添加字段并点击页面元素
12. 验证各字段选择器

### 预期结果
- 选择器生成准确
- 高亮显示清晰
- 手动调整响应流畅

## M-02: richText 预览渲染

### 验收场景
HTML 内容在预览对话框中的渲染效果

### 验收要点
1. **样式是否正确**
   - 段落、标题、列表等 HTML 标签应正确渲染
   - 样式应接近原始网页效果

2. **中文显示**
   - 中文内容应正确显示，无乱码
   - 字体应与网页一致

3. **图片显示**
   - 图片应正常加载显示
   - 图片大小应适应容器

### 验收步骤
1. 执行一个已配置好的爬虫任务
2. 进入内容管理页面
3. 点击某条包含 richText 的内容的"预览"按钮
4. 验证预览对话框中内容渲染效果

### 预期结果
- HTML 内容正确渲染
- 中文无乱码
- 图片正常显示

## M-03: Excel 打开体验

### 验收场景
导出的 Excel 文件在不同 Office 版本打开的兼容性

### 验收要点
1. **格式兼容**
   - 文件能正常打开，无格式错误提示
   - WPS、Office 2016、2019、365 均能正常打开

2. **列宽自适应**
   - 列宽应根据内容自动调整
   - 长文本不应被截断

3. **合并单元格（如有）**
   - 合并单元格应正确显示

### 验收步骤
1. 执行一个已配置好的爬虫任务，确保有内容数据
2. 进入内容管理页面
3. 点击"导出"按钮下载 Excel
4. 分别用以下软件打开：
   - WPS
   - Microsoft Office 2016
   - Microsoft Office 2019
   - Microsoft Office 365（如有）
5. 验证文件正常打开且格式正确

### 预期结果
- 各版本均能正常打开
- 列宽自适应
- 内容完整显示

## M-04: 真实网站稳定性

### 验收场景
爬取真实网站（如新浪体育）的长期稳定性

### 验收要点
1. **选择器是否失效**
   - 观察选择器是否能持续正确提取内容
   - 网站结构变化时是否能及时发现

2. **频率限制应对**
   - 爬虫是否能正确处理网站的频率限制
   - 是否有过早被封禁的情况

3. **错误处理是否友好**
   - 网络异常时错误提示是否清晰
   - 是否能区分不同类型的错误

### 验收步骤
1. 配置一个爬取新浪体育的任务
2. 启用定时调度（每小时执行一次）
3. 持续监控一周
4. 记录以下指标：
   - 成功次数 / 总次数
   - 选择器失效次数
   - 错误类型分布
   - 是否有频率限制提示

### 预期结果
- 成功率 > 95%
- 选择器未失效
- 错误处理合理

## M-05: UI 美观度

### 验收场景
页面布局、颜色、字体等视觉体验

### 验收要点
1. **整体风格一致**
   - 各页面风格应保持一致
   - 与设计稿是否一致

2. **响应式布局**
   - 在不同屏幕尺寸下是否正常显示
   - 是否有布局错乱

3. **交互反馈**
   - 按钮点击、hover 等状态是否有反馈
   - 加载状态是否清晰

### 验收步骤
1. 打开浏览器开发者工具
2. 分别在以下视口下检查：
   - 1920x1080（桌面）
   - 1366x768（笔记本）
   - 375x667（手机）
3. 检查以下页面：
   - 任务列表页
   - 任务配置页
   - 内容管理页
   - 内容编辑页
4. 验证布局、样式、交互

### 预期结果
- 风格一致美观
- 响应式布局正常
- 交互反馈清晰
